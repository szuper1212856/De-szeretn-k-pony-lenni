# De-szeretnékk-pony-lenni
Az eredeti szövege a dal "De szeretnék pony lenni"-nak
Ez az a dal egy remix, ha az eredetit szeretnéd megtekinteni, itt van a dalszöveg. 

De szeretnék páva lenni,
Főispánnak lánya lenni,
A kiskertbe sétálgatni,
Tejarózsát szakítgatni.

Sírhat az az édesanya,
Kinek fija csizmadia,
Azt se tudja, mely órába
Esik a csirizes tálba.

Előadoi: Szili Anna, Zsuberács Mária, Keresztes Rozália, Hollósi Teréz, Kovács Ágnes, Kovács Teréz
